Desafío
Implementar un programa que tome una cadena de paréntesis y determine si el orden de los paréntesis es válido. El programa debería devolver verdadero(true) si la cadena es válida, y falso(false) si no es válida.
Ejemplos

"()"               => true

")(()))"           => false

"("          => false

"(())((()())())" => true

Detalles

- Solo paréntesis. Cualquier otro carácter diferente producirá false.

- Desbalanceados significa que algún paréntesis no tiene pareja de cierre o apertura.

- '' representa a una cadena vacía ingresada en el recuadro.
